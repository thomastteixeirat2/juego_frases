Descripción

El Juego de Frases es un programa interactivo desarrollado en Python que requiere de la habilidad del usuario. El sistema muestra una frase al azar y el jugador debe copiarla exactamente en un tiempo y forma. El juego incorpora un sistema de vidas y registra los resultados en un archivo de texto.

Este proyecto fue desarrollado como parte de la Expo Técnica 2025 para demostrar dominio de programación estructurada.
Características principales

Selección aleatoria de frases.

Control de tiempo para cada intento.

Sistema de 3 vidas.

Validación exacta de la frase ingresada.

Registro de resultados en un archivo resultados.txt.
Conceptos de programación utilizados

Listas

Control de flujo (if, else, while)

Módulos estándar (random, time)

Entrada/salida de usuario

Manejo básico de archivos
